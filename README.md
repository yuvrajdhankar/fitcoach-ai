# 🏋️ FitCoach AI

🚀 A modern AI-powered fitness tracking and coaching web app.

## 🔥 Features

* 📊 Dashboard with analytics & progress tracking
* 🏋️ Workout logging system
* 🧠 AI workout plan generator
* 🍽️ Meal tracking
* 🔐 Firebase Authentication (Login system)
* ☁️ Cloud Firestore database integration
* 🌐 Fully deployable web app

## 🛠️ Tech Stack

* HTML, CSS, JavaScript
* Firebase (Auth + Firestore)
* GitHub + Netlify Deployment
